La iniciativa

Tracktweet es una iniciativa personal, un proyecto para descubrir, aprender y experimentar. Es una sencilla herramienta para ver tu timeline con los tweets geolocalizados ubicados en el mapa. Funciona consumiendo del API REST de Twitter, con los mapas de Leaflet (OpenStreetMap), Django y Twython una versatil libreria para conectar con el API de Twitter. Ademas jQuery (con algunos plugins), y el API de Google Maps para la codificación geográfica inversa para obtener las direcciones de los tweets geoposicionados
