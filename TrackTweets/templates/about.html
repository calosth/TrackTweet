La iniciativa

Tracktweet es una iniciativa personal, un proyecto para descubrir, aprender y experimentar. Soy @calosth el autor y con este experimento busco poner mi primer proyecto en la web. Es una sencilla herramienta para ver tu timeline y ver los tweets geolocalizados. Esta hecho por supuesto utilizando el API REST de Twitter, los mapas son de Leaflet , esta hecho en Django y Twython una versatil libreria para conectar con el API de Twitter.
